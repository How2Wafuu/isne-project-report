\begin{abstractTH}
โครงการนี้เสนอการออกแบบและการใช้งานระบบการจัดการข้อมูลและเหตุการณ์ด้านความปลอดภัย (SIEM) ที่ปรับแต่งให้ตรงตามข้อกำหนดเฉพาะของมาตรฐาน HAIT+ วัตถุประสงค์หลักคือการจัดหาโซลูชันแบบรวมศูนย์ที่มีประสิทธิภาพด้านต้นทุนสำหรับการตรวจสอบระบบสารสนเทศของโรงพยาบาล (HIS) และโครงสร้างพื้นฐานเครือข่ายที่สำคัญ วิธีการนี้เกี่ยวข้องกับการใช้งาน Wazuh ที่เป็นระบบ SIEM เป็นหลัก โดยกำหนดค่าให้รวบรวมบันทึกจากแหล่งข้อมูลที่สำคัญ รวมถึง Active Directory ไฟร์วอลล์ และอื่นๆ กฎการตรวจจับแบบกำหนดเองได้รับการพัฒนาขึ้นเพื่อให้ตรงกับข้อกำหนดของ HAIT+ โดยเฉพาะ โดยเน้นที่การใช้งานบัญชีผู้มีสิทธิ์พิเศษและความพยายามในการเข้าถึงโดยไม่ได้รับอนุญาตเพื่อให้มั่นใจในความสมบูรณ์ของข้อมูล
\end{abstractTH}

\begin{abstract}
This project proposes the design and implementation of a Security Information and Event Management (SIEM) system tailored to meet the specific requirements of the HAIT+ standard. The primary objective is to provide a cost-effective, centralized solution for monitoring critical hospital information systems (HIS) and network infrastructure. The methodology involves deploying Wazuh as the core SIEM system, configured to aggregate logs from essential sources including active directory, firewalls, and so on. Custom detection rules were developed to map specifically to HAIT+ requirements, focusing on privileged account usage and unauthorized access attempts to ensure data integrity.
\end{abstract}
